# sae: php frameworks

duration: 11 * 3h (maybe more, request from martin)

learn about mvc, rest and the laravel framework.

## demo project

```bash
# install dependencies
composer install

# start server
composer run-script serve

# (re-)create database
composer run-script seed
```
